Data Cleaning and Feature Extraction¶

Our EDA revealed several things about our dataset that we'll want to address before building our model. The decisions here are a combination of necessity, wanting to use a variety of techniques (this is a side project, after all), and wanting to improve our model. Namely, we've decided to:

  1. Drop vehicles that do not have a dealer price
  2. Remove outliers (we won't be dropping all outliers, just a couple 'extreme' outliers)
  3. Group together some vehicle makes
  4. Drop the 'Model' information
  5. Simplify exterior and interior color names to help group them together
  6. Split 'Transmission' into two different variables
  7. Standardize/Group some of the 'Drivetrain' values
  8. Fill in an 'Unknown' value in the 'Fuel_Type' column
  9. Create dummy variables for the 'Zipcode' data
  10. Impute the missing MPG and engine size data. We'll actually wait to do this until after splitting our data into a test/train subset.

There is probably more we can do, but that seems like a decent list for our purposes.

Here, we do steps 1 and 2 by dropping the desired vehicles.

There are a few ways we could group these together. We could try to group them by function (daily drivers, luxury, sports cars), by popularity (ranked by how many are in the dataset), or some other distinguisher. One metric might be the mean price of each Make, but it doesn't really seem like good practice to use the price as a way to group things (making a variable determined strictly by price, and then using that variable to predict price would be circular logic). In an effort to 1) avoid the circular logic and 2) have some sort of data-driven approach rather than random arbitrary choice, I have elected to group them based on how common they are in the dataset. This is likely to relate to price, but not specifically in a direct way, and makes it so I don't have to arbitrarily make the decisions.

# Identify the base color of the custom name colors
root_colors = ['silver','white','black', 'red', \
               'gray','yellow','orange', 'blue', \
              'metallic', 'pearl', 'grey', 'granite', \
              'clearcoat', 'scarlet', 'graphite', \
              'green', 'emerald', 'crimson', 'brown', 'charcoal', 'beige', 'ebony']
pattern = '|'.join(root_colors)


def pattern_searcher(search_str:str, search_list:str):
    if not isinstance(search_str, str):
        return 'NA'
    search_str = search_str.lower()
    search_obj = re.search(search_list, search_str)
    if search_obj :
        return_str = search_str[search_obj.start(): search_obj.end()]
        if return_str == "pearl":
            return_str = "white"
        elif return_str == "grey" or return_str == "granite":
            return_str = "gray"
        elif return_str == "clearcoat" or return_str == "graphite":
            return_str = "silver"
        elif return_str == "scarlet" or return_str == "crimson":
            return_str = "red"
        elif return_str == "emerald":
            return_str = "green"
        elif return_str == 'charcoal' or return_str == 'ebony':
            return_str = "black"
        elif return_str == 'beige':
            return_str = "brown"
        
        
    else:
        return_str = 'OTHER'
    return return_str

def transmission_pattern_searcher(search_str:str):
    if not isinstance(search_str, str):
        return 'NA'
    search_str = search_str.lower()
    search_obj = re.search("auto", search_str)
    if search_obj:
        tran_type = 'automatic'
    else:
        search_obj = re.search("variable", search_str)
        if search_obj:
            tran_type = "variable"
        else:
            tran_type = 'manual'
    return tran_type

def gears_searcher(search_str:str):
    if not isinstance(search_str, str):
        return 'NA'
    gears_obj = re.search("[\d]+", search_str)
    if gears_obj:
        gears = re.search("[\d]+", search_str)[0]
    else:
        gears = 'NA'
    return gears
